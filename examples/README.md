# Examples

This directory contains example repositories produced by Gitx.

Examples exist only for reference and demonstration.
They are not templates.
They are not starting points.

---

## Purpose

The purpose of this directory is to:

- Show the shape of generated repositories
- Demonstrate compliance with output contracts
- Provide concrete reference artifacts

Each example represents a finished result.

---

## Rules

- Examples must be immutable once published
- No manual edits are allowed
- Each example must conform to the output repository contract

If an example diverges from documentation,
the example is considered invalid.

---

## Status

This directory may be empty.

Examples will be added only after
the generation system is finalized.

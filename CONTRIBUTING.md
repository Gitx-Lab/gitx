# Contributing to Gitx

Thanks for your interest in Gitx.

At this stage, Gitx is a specification-first project. The repository primarily
contains design documents, contracts, and system definitions.

---

## Contribution Status

Gitx is currently in a controlled development phase.

- External code contributions are not accepted yet
- Documentation feedback may be requested in the future
- All changes are governed by the Gitx core maintainers

This file exists to clearly define expectations and prevent ambiguity.

---

## Allowed Contributions (Future)

When contributions are opened, they may include:

- Documentation improvements
- Specification clarifications
- Examples of generated projects
- Tooling or SDK proposals

All contributions will be reviewed against existing specifications.

---

## Rules

- Do not submit pull requests without prior approval
- Do not modify core specifications unless explicitly requested
- Do not introduce implementation code unless the scope is defined

Gitx prioritizes consistency and clarity over speed.

---

## Communication

Contribution guidelines may evolve as the project matures.
This file will be updated accordingly.
